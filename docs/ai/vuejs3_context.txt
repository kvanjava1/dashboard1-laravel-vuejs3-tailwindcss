# **Project Architecture Note: Compound Components Pattern**

## **Pattern Used: Compound Components Pattern**
This project implements the **Compound Components Pattern** (also known as Component Composition Pattern) for building reusable, maintainable UI components.

---

## **This Project is using Compound Components Pattern**
A design pattern where multiple Vue components work together to form a complete UI feature, sharing state implicitly while allowing flexible composition. Each component handles a specific part of the functionality and styling.

## **Key Characteristics:**
1. **Multiple collaborating components** that form a complete feature
2. **Implicit state sharing** through Vue's provide/inject or slots
3. **Declarative API** that reads like HTML/JSX
4. **Flexible composition** - users can arrange components as needed
5. **Single responsibility** - each component handles one specific role

---

## **Implementation in This Project:**

### **Table Component Structure:**
```
SimpleTable/
├── SimpleTable.vue          (Root container - provides styling context)
├── SimpleTableHead.vue      (Header section)
├── SimpleTableHeadRow.vue   (Header row)
├── SimpleTableHeadCol.vue   (Header column with width control)
├── SimpleTableBody.vue      (Body section)
├── SimpleTableBodyRow.vue   (Body row with hover effects)
└── SimpleTableBodyCol.vue   (Body column with width control)
```

### **Example Usage:**
```vue

    <SimpleTableHead>
        <SimpleTableHeadRow>
            <SimpleTableHeadCol width="80px">

            </SimpleTableHeadCol>
            <SimpleTableHeadCol>

            <SimpleTableHeadCol width="120px">
                Role
            </SimpleTableHeadCol>
        </SimpleTableHeadRow>
    </SimpleTableHead>
    
    <SimpleTableBody>
        <SimpleTableBodyRow v-for="user in users" :key="user.id">
            <SimpleTableBodyCol width="80px">
                {{ index + 1 }}
            </SimpleTableBodyCol>
            <SimpleTableBodyCol>
                {{ user.name }}
            </SimpleTableBodyCol>
            <SimpleTableBodyCol width="120px">

            </SimpleTableBodyCol>
        </SimpleTableBodyRow>
    </SimpleTableBody>
```
